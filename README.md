# Store
-A program used to checkout items

The program will display a list of items that can be checked out. By typing in the number corresponding to the item in the list, the item
will be added. The user can continue to add as many items as they want until they type "0". Which will then calculate the total of items
and show the user how much everything will cost with the subtotal, tax, and overall total that inludes the cost of the items and the sales
tax.
